# Eunomia Documentation

Welcome to the Eunomia room scheduling and management system documentation.

## Overview

Eunomia is a comprehensive room reservation and management system designed for conferences, meetings, and events. It provides tools for:

- **Room Reservations**: Book and manage conference rooms and meeting spaces
- **Schedule Management**: Organize conference schedules and session timing  
- **Side Events**: Coordinate side events and special sessions
- **CCTV Management**: Monitor camera feeds and video surveillance
- **Administration**: User management and system configuration
